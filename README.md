# TabLens

TabLens is a Chrome extension that helps you make sense of your browser chaos. See what tabs are eating your attention, spot duplicates, find forgotten tabs, and group the rest — all in one clean dashboard.

## Features

- **Overview Dashboard** — Net growth, most visited tabs, oldest survivors, duplicates, and 14-day trend
- **All Tabs View** — Search, filter, and close tabs with undo support
- **Distraction Alert** — Surface tabs you switch to most often
- **Zombie Tabs** — Find tabs open for hours with no activity
- **Smart Grouping** — One-click suggestions based on domain and keywords
- **Duplicate Detection** — Clean up redundant tabs instantly

## Install

1. Clone or download this repo
2. Open Chrome → `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the project folder
